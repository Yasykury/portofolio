# Portfolio media — circle-media

Drop your work here: images (.jpg .png .webp .gif) and videos (.mp4 .webm .mov).
Everything appears AUTOMATICALLY on /work/circle-media — with hover preview, auto-play
videos, and a click-to-enlarge lightbox. The first image also becomes the
hover thumbnail on the home page.

Tips:
- Prefix filenames with numbers (01-..., 02-...) to control the order.
- After adding VIDEOS, run "npm run optimize" once to compress them for fast
  loading (safe to re-run; skips already-optimized files).
- Publish:  git add .   →   git commit -m "Tambah karya circle-media"   →   git push
