# Project thumbnails

Drop your real project images / video stills here, then point each project at its
file in `lib/site.ts` by setting the `image` field, e.g.:

```ts
{
  title: "PT Dupoin Futures Indonesia (GRW)",
  // ...
  image: "/projects/dupoin.jpg",
}
```

Tips:
- Use a **16:10** ratio for a perfect fit (e.g. 1280×800).
- `.jpg`/`.webp` for photos/stills, `.png` for graphics.
- Until a project has an `image`, it shows the gradient placeholder cover automatically.
